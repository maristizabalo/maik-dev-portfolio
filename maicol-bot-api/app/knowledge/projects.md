# Proyectos

- LemonVox: dashboard de llamadas, grabaciones y CDR sobre Asterisk, FreePBX e Issabel. Stack: Go, Svelte, Docker, Caddy, MySQL y Zitadel.
- SmartVIP: modulo contable para comprobantes, ventas, notas credito y SQL. Stack: PHP, MySQL y SQL.
- INVERTAXI: plataforma de revision y fallos vehiculares con React, Python, automatizacion y analisis de datos.
- DADEP: experiencia institucional con React, Next.js, Python, Node.js, PostgreSQL, Oracle, AWS y Firebase.
- Kroco: plataforma de torneos con agente de voz e IA.
- Bien Company: backend PHP con AWS SQS, Lambda y GCP Firestore.
